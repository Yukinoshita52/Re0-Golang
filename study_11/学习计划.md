# 第十一阶段学习计划：小型后端项目实践

## 阶段目标

完成这一阶段后，你应该能围绕“小型后端项目实践”独立创建 Go 项目、手敲示例代码、运行验证命令，并用自己的话复盘关键概念。

本阶段的代码文件由你自己在 GoLand 中创建。本仓库只预先提供这份学习计划。

## 准备确认

开始前确认：

- 已使用 GoLand 创建或打开 `study_11` 目录。
- 已准备在 `study_11` 目录下初始化 Go module。
- 能在 GoLand Terminal 中执行 Go 命令。

## 需要你手动创建的文件

- `go.mod`
- `cmd/server/main.go`
- `internal/...`
- `README.md`

## 学习重点

- 项目结构
- 配置
- 日志
- 错误处理
- HTTP API
- MySQL 与 Redis 集成
- 测试与可维护性
