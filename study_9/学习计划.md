# 第九阶段学习计划：HTTP 服务入门

## 阶段目标

完成这一阶段后，你应该能围绕“HTTP 服务入门”独立创建 Go 项目、手敲示例代码、运行验证命令，并用自己的话复盘关键概念。

本阶段的代码文件由你自己在 GoLand 中创建。本仓库只预先提供这份学习计划。

## 准备确认

开始前确认：

- 已使用 GoLand 创建或打开 `study_9` 目录。
- 已准备在 `study_9` 目录下初始化 Go module。
- 能在 GoLand Terminal 中执行 Go 命令。

## 需要你手动创建的文件

- `go.mod`
- `main.go`
- `main_test.go`

## 学习重点

- net/http 服务端
- 路由基础
- 请求参数解析
- JSON 响应
- 中间件思想
- 优雅关闭入门
