# Go 和 Golang 这两种叫法有区别吗？

## 简短结论

官方名称是 **Go**。

**Golang** 不是官方名称，更像是社区里的俗称，常用于搜索、标签、域名和避免歧义。

## 详细说明

`Go` 是这门语言的正式名称。写文档、简历、技术文章或项目说明时，推荐使用“Go 语言”。

`Golang` 这个叫法流行起来，主要是因为 `go` 这个词太通用，搜索时容易混入无关内容。同时，Go 早期官网域名是 `golang.org`，所以很多资料、标签和社区内容会使用 `golang` 来指代 Go 语言。

因此，两者通常指的是同一门语言，只是使用场景不同：

| 叫法 | 使用场景 |
| --- | --- |
| Go | 官方名称，适合正式文档、文章、简历、项目说明 |
| Golang | 社区俗称，适合搜索关键词、标签、域名或避免歧义 |

## 记忆点

- 正式写法：Go。
- 搜索资料：可以用 Golang。
- 文档和简历：建议写 Go 语言。
- 两种叫法通常指同一门语言，不是两门不同技术。
