# 知识问答

本目录用于记录学习 Go 过程中需要长期沉淀的小问题。

适合记录的问题包括：

- 小概念：例如 `defer`、`goroutine`、`channel` 是什么。
- 小区别：例如 Go 和 Golang 的叫法区别、数组和切片的区别。
- 小术语：例如 package、module、workspace、receiver 等。
- 易混点：例如值传递、指针、接口实现、错误处理习惯。

## 记录规则

- 只有用户明确要求“记录”“沉淀”“写入文档”等，才将问答整理到本目录。
- 文档统一使用中文编写。
- 单个问题可以独立成文，也可以按主题归类。
- 内容应尽量短小清晰，便于复习。
- 内容默认专注 Go 语言本身，不固定加入 Java 对比。
- 只有当 Java 经验可能造成误解时，才补充“易误解点”或“与 Java 经验的差异”说明。

## 建议格式

```markdown
# 问题标题

## 简短结论

先用几句话给出直接答案。

## 详细说明

展开解释背景、区别或常见误区。

## 易误解点

只有在需要澄清常见误解时才添加本节；如果误解来自 Java 经验，可以在这里说明。

## 记忆点

用简短列表记录复习重点。
```
